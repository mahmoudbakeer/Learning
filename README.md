## What's Inside?

Here is a breakdown of the directories in this repository and what you will find inside each one:

###  `Books`
A collection of notes, summaries, and code implementations inspired by the software engineering books and technical documentation I am currently reading. Continuous learning is key!

###  `C#_Advanced`
Where the heavy lifting happens. This folder contains advanced exercises and deep dives into C# mechanics.
* **Current Focus:** Collections, `SortedSets`, `HashSets`, and advanced LINQ queries to write highly optimized and clean code.

###  `EF/EF001`
My practical implementations using **Entity Framework Core**. 
* **Current Focus:** Building robust Data Access Layers, database first/code first approaches, migrating schemas, and managing database contexts (like `AppDbContext`).

###  `T-Sql`
A dedicated space for database logic. Writing efficient queries is just as important as writing clean backend code.
* **Current Focus:** Revisions on SQL fundamentals, complex joins, stored procedures, and integrating SQL seamlessly with C# applications.

###  `Vim`
Developer productivity matters! This folder tracks my journey in mastering Vim keybindings and optimizing my development environment.
* **Current Focus:** Exercises combining Vim muscle memory with LINQ and T-SQL development, ensuring I can navigate and edit code at the speed of thought.

###  `WindowFunction`
Advanced database querying techniques.
* **Current Focus:** Extensive practice examples utilizing T-SQL Window Functions (`OVER()`, `ROW_NUMBER()`, `RANK()`, etc.) for complex data analysis and manipulation directly within the database engine.
